# SmartTradingBot
Bot de Trading Autônomo para o Mercado de Futuros com Análise Técnica e Gestão de Risco. Conectado à API da Bybit, o bot realiza operações automatizadas com base em indicadores técnicos e padrões de vela, oferecendo notificações em tempo real via WhatsApp ou Telegram. Projeto modular e escalável, com foco em performance e segurança.
